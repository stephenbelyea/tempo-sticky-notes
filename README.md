# Tempo Sticky Notes - React TS App

## Development

### Requirements

- [NPM + Node](https://nodejs.org/en/download)
- [NVM](https://nodejs.org/en/download/package-manager/all#nvm)

### Run Locally

1. Clone the repo and navigate to its root
2. Run `nvm use`
3. Run `npm install`
4. Run `npm start`
